# Project Echo: Local AI English Tutor (S23 Ultra Edition)

Um tutor de inglês que roda 100% offline no Android via Termux, utilizando Llama-3 para correção gramatical e Piper TTS para voz.

## Requisitos
- Android com Termux instalado (F-Droid version recomendada).
- Mínimo 8GB de RAM (Recomendado 12GB).
- Processador Snapdragon 8 Gen 2 ou superior (para boa performance).

1. Abra o Termux.

   pkg update -y && pkg upgrade -y
   pkg install git -y
